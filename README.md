# Python Quiz App

A simple and interactive Quiz Application built using Python.  
This project asks multiple-choice questions, validates user answers, and calculates the final score instantly.

---

## Features

- Multiple-choice quiz system
- Instant answer validation
- Final score calculation
- Beginner-friendly Python project
- Clean command-line interface

---

## Tech Stack

- Python

---

## Project Structure

```bash
├── quiz_app.py
└── README.md
```

---

## How It Works

1. The program displays a question.
2. Multiple options are shown.
3. User enters the answer.
4. The app checks whether the answer is correct.
5. Final score is displayed at the end.

---

## Example Output

```text
What is the capital of India?

A. Mumbai
B. Delhi
C. Kolkata
D. Chennai

Enter your answer (A/B/C/D): B

✅ Correct!
```
